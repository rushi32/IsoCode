# Contributing to IsoCode

- Report bugs, suggest changes, or propose features via Issues.
- Code changes go through Pull Requests ([GitHub Flow](https://guides.github.com/introduction/flow/)).

1. Fork and branch from `main`.
2. Add tests for new code; update docs if you change APIs.
3. Run the test suite and lint.
4. Open a PR.

## Development Setup

1. Clone the repo.
2. Run `npm install` in the root and in `extension/`.
3. Create a `.env` file based on configuration needs.
4. Run `node server/index.js` to start the backend.
5. Open `extension/` in VS Code and press F5 to launch the extension.
